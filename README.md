# Restify Customer API

> RESTful API built with Node.js, Restify, Mongoose and JWT. It uses 'jsonwebtoken' to create the token and restify-jwt-community to protect routes.

## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:3000
npm start
```

## API Endpoints

#### Customer Routes

- GET /customers
- GET /customers/:id
- POST (Protected) /customers
- PUT (Protected) /customers/:id
- DELETE (Protected) /customers/:id

#### Genre Routes

- GET /api/genres
- POST /api/genres
- PUT /api/genres/:id
- DELETE /api/genres/:id

#### Book Routes

- GET /api/books
- POST /api/books
- PUT /api/books/:id
- DELETE /api/books/:id

#### User Routes

- POST /register
- POST /auth
